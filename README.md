
# Hangman Game 🎮

An interactive Hangman game built in **Java** with a clean GUI using **Swing**. The project demonstrates core programming concepts such as event handling, string manipulation, and GUI design, while providing a fun word-guessing experience.

---

## ✨ Features
- Dynamic word display with underscores for hidden letters
- Clickable alphabet buttons for guessing
- Progressive hangman visuals drawn with Swing graphics
- Color-coded feedback for correct/incorrect guesses
- Restart option to play multiple rounds
- Score tracking for wins and losses

---

## 🛠️ Tech Stack
- **Language:** Java
- **GUI Framework:** Swing
- **IDE:** Eclipse / IntelliJ IDEA
- **Version Control:** GitHub
